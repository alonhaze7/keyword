📊 Interactive Preview

This is a static HTML preview showing the top 3 widgets. The full dashboard includes 10 interactive widgets with:

  • Waste Trend Chart (30-day time series)
  • Waste by Match Type (pie chart)
  • Top Waste Categories (bar chart)
  • Keyword Performance Matrix (scatter plot)
  • Intent Mismatch Table (sortable, filterable)
  • Search Term Cloud (interactive word cloud)
  • Cost Savings Calculator (3 scenarios)

To see the full interactive dashboard:

  1. Install Node.js if not already installed
  2. Navigate to the project folder in terminal
  3. Run: npm install && npm run dev
  4. Open browser to http://localhost:3000
